# GPT Image V2 Prompt Crafter

一个面向 **GPT Image V2（OpenAI）** 的 Prompt 生成 Skill。

## 包含内容

- `SKILL.md`：Skill 主说明
- `prompts/GPT-Image-V2-Prompt-Template-Handbook.md`：完整模板手册
- `examples/quick-social-live.md`：自然语言直出示例
- `examples/precision-saas-hero.md`：高精度结构化示例

## 使用方式

- 简单需求 → 走 Quick 模式
- 强控制需求 → 走 Precision 模式
- 不确定时 → 默认 Quick；用户要求“结构化 / 高控制”时切到 Precision
